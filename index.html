V2ray easy to use (V0.9.5 RC)

  • 这是一个简单的V2ray部署教程

  • V2ray+Caddy一键脚本（WebSocket+TLS+Website）：

  • 一键安装脚本说明：

    • 本一键脚本仅支持 CentOS/Debian/Ubuntu ！

    This one-click script only supports CentOS / Debian / Ubuntu !

  • 若您使用本一键脚本进行搭建，那么您采用的模板是ws+tls+web。

    • 地址(Address)：即您的域名

    端口(Port)：443

    用户ID(ID)：由系统生成

    额外ID(AlterId)：100

    加密方式(Security)：none

    传输协议(Network)：ws

    别名(Remarks)：随您的喜好填写任意字符

    伪装类型(Type）：none

    伪装域名/其他项：/tmp/video

    底层传输安全(TLS)：tls


  • 一键安装脚本：

    • wget https://www.v2ray-install.ml/v2ray-go.sh && bash v2ray-go.sh && rm -rf v2ray-go.sh

  • 一键卸载脚本：

    • wget https://www.v2ray-install.ml/v2ray-remove.sh && bash v2ray-remove.sh && rm -rf v2ray-remove.sh

  • 一键重装脚本：

    • wget https://www.v2ray-install.ml/v2ray-reborn.sh && bash v2ray-reborn.sh && rm -rf v2ray-reborn.sh

  • 手动配置说明：

    • 以下方法仅支持 CentOS/Debian/Ubuntu ！

    The following methods only support CentOS / Debian / Ubuntu !

  • 若您完全安装下文所述进行安装，那么您采用的模板是ws+tls+web。

    • 地址(Address)：即您的域名

    端口(Port)：443

    用户ID(ID)：由系统生成

    额外ID(AlterId)：100

    加密方式(Security)：none

    传输协议(Network)：ws

    别名(Remarks)：随您的喜好填写任意字符

    伪装类型(Type）：none

    伪装域名/其他项：/tmp/video

    底层传输安全(TLS)：tls


  • 1.注册域名，参考：

    • 教你申请.tk/.ml/.cf/.gq/.ga等免费域名

    使用 CloudXNS 接管 Freenom 的免费域名解析，加快国内生效速度！

  • 2.检查OS兼容情况，并更新系统组件：

    • wget https://www.v2ray-install.ml/checkos.sh && bash checkos.sh

  • 3.下载并安装V2ray：

    • curl https://www.v2ray-install.ml/install-release.sh | bash

  • 4.下载配置文件，并启动V2ray：

    • rm /etc/v2ray/config.json && wget -qO /etc/v2ray/config.json "https://www.v2ray-install.ml/config.json" && service v2ray restart

  • 5.安装Caddy：

    • curl https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/caddy_install.sh | bash

  • 6.下载Caddy配置文件：

    • wget -qO /usr/local/caddy/Caddyfile "https://www.v2ray-install.ml/Caddyfile"

  • 7.创建wwwroot目录，并拉取混淆网页：

    • cd / && mkdir www && cd www && mkdir wwwroot && cd wwwroot && wget https://www.v2ray-install.ml/webpage.zip && unzip webpage.zip && rm webpage.zip

  • 8.按提示替换Caddy配置文件中的汉字：

    • vi /usr/local/caddy/Caddyfile

    （附：Vim工具使用教程：Linux 文本编辑器 Vim 真 · 简单使用教程）

  • 9.修改完毕后，重启Caddy，并返回到root目录：

    • service caddy restart && cd

  • 10.生成UUID&Port，并返回配置信息：

    • curl https://www.v2ray-install.ml/change-info.sh | bash

  • 11.安装BBR加速TCP流量（可选）：

    • curl https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/bbr.sh | bash

  • 12.下载V2ray客户端：客户端与服务 · Project V 官方网站
