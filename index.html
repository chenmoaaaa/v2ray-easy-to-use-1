V2ray easy to use (V0.7 Beta)

  • 这是一个V2ray部署教程，可以简单快速的在您的vps部署好一个可以使用的V2ray程序。

  • V2ray+Caddy一键安装脚本（WebSocket+TLS+Website）：

    • cd && mkdir v2ray-go && cd v2ray-go && wget https://v2ray-install.netlify.com/v2ray-go.sh && chmod +x v2ray-go.sh && bash v2ray-go.sh

  • 若安装过程中提示如下信息，请输入Y并回车。

    • Do you want to continue? [Y/n]

  • 安装过程中涉及到使用Vim工具，具体使用教程请参考：Linux 文本编辑器 Vim 真 · 简单使用教程

  • 本一键脚本仅支持 CentOS/Debian/Ubuntu ！

  • This one-click script only supports CentOS / Debian / Ubuntu !

  • 可能是自一键脚本诞生以来做得最垃圾的一个！

  • 本一键脚本基于https://v2ray.liping.ga修改。

  • 以下方法仅支持 Debian/Ubuntu ！

  • The following methods only support Debian / Ubuntu !

  • 手动配置说明：

    • 若您完全安装下文所述进行安装，那么您采用的模板是ws+tls+web。

    地址(Address)：即您的域名

    端口(Port)：443

    用户ID(ID)：由系统生成

    额外ID(AlterId)：100

    加密方式(Security)：none

    传输协议(Network)：ws

    别名(Remarks)：随您的喜好填写任意字符

    伪装类型(Type）：none

    伪装域名/其他项：/tmp/video

    底层传输安全(TLS)：tls


  • 1.注册域名，参考：

    • 教你申请.tk/.ml/.cf/.gq/.ga等免费域名

    使用 CloudXNS 接管 Freenom 的免费域名解析，加快国内生效速度！

  • 2.更新、下载所需要的组件：

    • apt-get update && apt-get install wget

  • 3.下载并安装V2ray：

    • cd && mkdir v2ray && cd v2ray && wget https://v2ray-install.netlify.com/install-release.sh && bash install-release.sh

  • 4.下载配置文件，并启动V2ray：

    • rm /etc/v2ray/config.json && wget -qO /etc/v2ray/config.json "https://v2ray-install.netlify.com/config.json" && service v2ray restart

  • 5.安装Caddy：

    • cd && mkdir caddy && cd caddy && wget https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/caddy_install.sh && chmod +x caddy_install.sh && bash caddy_install.sh

  • 6.下载Caddy配置文件：

    • wget -qO /usr/local/caddy/Caddyfile "https://v2ray-install.netlify.com/Caddyfile"

  • 7.创建wwwroot目录，并拉取混淆网页：

    • cd / && mkdir www && cd www && mkdir wwwroot && cd wwwroot && apt-get install unzip && wget https://v2ray-install.netlify.com/webpage.zip && unzip webpage.zip && rm webpage.zip

  • 8.按提示替换Caddy配置文件中的汉字：

    • apt-get install vim && vi /usr/local/caddy/Caddyfile

    （附：Vim工具使用教程：Linux 文本编辑器 Vim 真 · 简单使用教程）

  • 9.修改完毕后，重启Caddy，并返回到root目录：

    • service caddy restart && cd

  • 10.生成UUID&Port，并返回配置信息：

    • cd && cd v2ray && wget https://v2ray-install.netlify.com/checkinfo.sh

  • 11.安装BBR加速TCP流量（可选）：

    • cd && mkdir bbr && cd bbr && wget https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/bbr.sh && chmod +x bbr.sh && bash bbr.sh

  • 12.下载V2ray客户端：客户端与服务 · Project V 官方网站
